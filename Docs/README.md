# ChessKnight landing page

# Folder Structure
```
.index.html         Static page, the landing itself
./WegGL/            WebGL builds of unity player
./TemplateData/     Landing page data
  ./images/         Contains images, icons etc
  ./script.js       Javascript for processing landing
  ./style.css       CSS for landing
./TemplateSource/   Source files for TemplateData
  ./scss/           SCSS source files for TemplateData/style.css
```
